# Arch Linux Interactive Setup Script

A Q&A-style installation script for Arch Linux.

## Features
- Step-by-step disk partitioning (fdisk/parted/cfdisk)
- Base package installation with pacstrap
- fstab generation
- Timezone, locale, hostname setup
- Optional Desktop/WM installation (KDE, GNOME, XFCE, i3, Sway)
- User creation with sudo
- Easy to start
## Usage
```bash
chmod +x setup.sh
./setup.sh
